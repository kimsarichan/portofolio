hybrid recommendation
Rekomendasi hybrid yang menggabungkan metode knowledge based recommender system dan user based collaborative filtering yang merekomendasikan item berdasarkan kesamaan dengan user lain 
yang didapat dari interaksi user dan data eksplisit dari profil user, dalam kasus ini item merupakan data pekerjaan dari website xing *data tidak untuk dibagikan*

jaccard coefficient and pearson corellation
Jaccard coefficient adalah teknik mengukur similiarity berdasarkan kesamaan set sampel dari tiap item
Pearson correlation adalah teknik mengukur similiarity bedasarkan kesamaan rating antar item
